# Project 2 — Phishing Email Analysis Project

## Overview
In this project my goal is to analyze different types of phishing emails, which departments they are targeting and how employees are responding. 
I also want to analyze the email domains and see if there is any correlation between domains and successful attacks (defined as a click or download).
Then to provide insights and recommendations in alignment with MITRE ATT&CK and NIST.

## Objectives
- Build an excutive dashboard that clearly illustrates the data
- Comment on the data 
- Write a Summary Report

## Dataset
5000 logs were synthetically created with a Python script Gemini helped me with.

## Tools Used
- Colab, Visual Studio Code, Python, Microsoft Excel
- PowerQuery
- PivotTables and PivotCharts
- Conditional formatting
- IFERROR
- MAXIFS, AVERAGEIFS, COUNTIFS

## What I learned
- I was much more organized on this second project
- I really wanted to connect cyber princples and frameworks with the data analysis
- One really useful lesson was looking not just at averages but maximum values because the data showed 3 different things (Finance was bombarded with attacks, Executive made some)
- Dashboard design is not my forte but I think I asked some good questions of the data
