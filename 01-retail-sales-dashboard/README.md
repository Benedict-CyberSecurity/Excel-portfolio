# Project 1 — Retail Sales Dashboard

## Overview
This project analyzes retail sales performance across products, regions, and time periods using Excel.

## Objectives
- Understand sales performance trends
- Identify high and low performing products and regions
- Build a dashboard for key business metrics

## Dataset
Simulated retail sales data (to be replaced/updated as project develops).

## Tools Used
- Microsoft Excel
- PivotTables
- PivotCharts
- XLOOKUP
- SUMIFS
- Conditional formatting

## Key Questions
- Which regions generate the highest revenue?
- Which products are most/least profitable?
- How do sales trends change over time?

In progress
